Sık Sorulan Sorular

Merak Ettiklerin

Platform ücretsiz mi?
Evet, platform bir ders projesi olarak geliştirilmiştir ve tamamen ücretsizdir. Ticari amaç taşımamaktadır.
Hangi cihazlarda çalışır?
Modern tarayıcı destekleyen tüm cihazlarda çalışır. Mobil uyumlu (responsive) tasarımı sayesinde telefon ve tabletlerde de sorunsuz kullanılabilir.
API anahtarı gerekiyor mu?
Mapbox ve SunCalc API entegrasyonu için kişisel API anahtarları gereklidir. Kurulum talimatları README dosyasında açıklanmıştır.
Verilerim güvende mi?
Kaydettiğin lokasyon ve notlar yalnızca kendi veritabanında saklanır. Herhangi bir üçüncü tarafla paylaşılmaz.
Projeye katkıda bulunabilir miyim?
GitHub üzerinden projeye ulaşarak pull request gönderebilirsin. Her türlü katkıya açığız.
Başka lensler için de öneri var mı?
Platform 135mm odaklı olsa da 85mm, 50mm ve 24mm gibi lensler için de ekipman önerileri ve notlar eklenebilmektedir.
