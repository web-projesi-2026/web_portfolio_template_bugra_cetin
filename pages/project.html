Sistemin
Çalışma Mantığı

Dış-Mekan-Çekimi, üç katmanlı bir web mimarisi üzerine inşa edilmiştir. Kullanıcı arayüzü HTML/CSS/JS ile geliştirilmiş; iş mantığı PHP veya C# ile sunucu tarafında çalışmakta, veriler ise SQL veritabanında saklanmaktadır.

Harita ve ışık hesaplama özellikleri harici API'lere bağlıdır. Mapbox GL JS ile interaktif harita sunulurken, SunCalc API ile GPS koordinatına özel dakika hassasiyetinde güneş verileri çekilmektedir.

Kullanıcı kayıtları, lokasyon notları ve referans fotoğraflar ilişkisel veritabanında tutularak oturumlar arasında kalıcı erişim sağlanmaktadır.

API Entegrasyonları

Harici Servisler

🗺
Mapbox GL JS
İnteraktif Harita & Pin Yönetimi
Kullanıcıların çekim noktalarını harita üzerinde işaretlemesini, pin ekleyip düzenlemesini ve kategorilere göre filtrelemesini sağlar.
GET /styles/v1/mapbox/dark-v11
🌅
SunCalc API
Güneş Konumu & Altın Saat Hesabı
Enlem/boylam koordinatları ve tarih bilgisiyle gün doğumu, gün batımı, sabah/akşam altın saat zamanlarını dakika hassasiyetinde hesaplar.
SunCalc.getTimes(date, lat, lng)
🌤
OpenWeatherMap
Anlık Hava Durumu & Tahmin
Seçilen lokasyonun anlık bulutluluk oranı, rüzgar hızı ve hava durumu tahminini çekerek çekim koşullarını önceden değerlendirmeye yardımcı olur.
GET /data/2.5/weather?lat=&lon=
📡
Geolocation API
Anlık Konum Tespiti
Tarayıcı Geolocation API'si aracılığıyla kullanıcının mevcut GPS konumunu tespit eder ve haritayı otomatik olarak mevcut konuma odaklar.
navigator.geolocation.getCurrentPosition()
